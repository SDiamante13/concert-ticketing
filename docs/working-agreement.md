# CollabAI

## Use AI for system 1 thinking

fast, automatic, intuitive, effortless, emotional.           
Pattern-matches and jumps to conclusions.

- generating options
- finding ideas
- writing code

## Using humans for system 2 thinking

slow, deliberate, analytical, effortful. Handles logic,         
calculation, and complex decisions.

- driver is summarizing output of the agent as we go
- making the difficult decisions as a group of humans
